# SocialApp

A modern, high-performance social media experience built with Next.js, Framer Motion, and Prisma.

## Features

- **Dynamic Feed**: Real-time posts and interactions.
- **Stories**: Animated story rings and full-screen story viewer.
- **Reels**: Immersive, edge-to-edge vertical video experience with real-time comments.
- **Live Streams**: High-quality streaming with interactive chat and theater mode.
- **Responsive Design**: Optimized for mobile, tablet, and desktop with premium aesthetics.
- **Dark Mode**: Fully implemented design system supporting dark and light themes.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS / Vanilla CSS
- **Animations**: Framer Motion
- **Database**: Prisma with SQLite
- **Icons**: Lucide React
