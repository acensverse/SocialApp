## âš™ï¸ Installation & Setup

Get up and running locally in just a few steps:

### 1. Requirements
Ensure you have **Node.js 18+** installed on your system.

### 2. Install Dependencies
```bash
npm install
```

### 3. Database Setup
SocialApp uses Prisma. Initialize your database schema:
```bash
npx prisma generate
npx prisma db push
```

### 4. Configuration
Create a `.env` file in the root directory based on `.env.example` and add your environment variables:
- `DATABASE_URL`: Your database connection string.
- `NEXTAUTH_SECRET`: Secret for session encryption.
- `AUTH_GOOGLE_ID` / `AUTH_GOOGLE_SECRET`: Google OAuth credentials.

### 5. Run the Developer Server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) to view the app.
